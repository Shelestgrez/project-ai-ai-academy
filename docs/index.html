Полное приложение — на Render

Важно: GitHub Pages — только витрина. Уроки, кабинет, XP и тесты — в полном приложении на localhost:5000 или на Render (*.onrender.com).

Быстрый запуск на Windows

  1. Установите Python 3.11+ и включите «Add Python to PATH».
  2. Клонируйте репозиторий: git clone https://github.com/Shelestgrez/project-ai-ai-academy.git
  3. В папке проекта запустите start_ai_academy.bat (создаётся .venv, ставятся зависимости, стартует сервер).
  4. Откройте в браузере http://127.0.0.1:5000. Ярлык на рабочий стол: create_desktop_shortcut.bat.

В сети (Render)

В репозитории есть render.yaml для Blueprint на Render: поднимаются веб-сервис и PostgreSQL, чтобы регистрация и прогресс сохранялись между перезапусками. После деплоя откройте выданный Render URL (вида *.onrender.com).
